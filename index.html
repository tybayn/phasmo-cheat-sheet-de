Voice Commands

Voice commands allow you to control the cheat sheet via your voice. Once you hit enable, the cheat sheet will begin listening to your voice commands and will persist between resets until you disable it. It will by default read from your default microphone, so ensure that it is set correctly. The voice recognition will continue to work even if the page is not in focus and you are using the same microphone in-game.
Evidence:
  • "number of evidence <num>" or "difficulty <num>" -> Set the number of evidence
  • "evidence <evidence>" -> Mark as a found evidence
  • "evidence not <evidence>" -> Mark as not an evidence
  • "evidence clear <evidence>" -> Reset state of an evidence
  • "monkey paw <evidence>" -> This will mark the evidence off with monkey paw logic
Speed:
  • "speed <speed>" -> Mark specified speed
  • "speed clear <speed>" -> Reset state of specified speed
  • "speed [has] line of sight" -> Mark having standard line-of-sight speed up
  • "speed not line of sight" -> Mark no standard line-of-sight speed up
  • "speed clear line of sight" -> Reset line-of-sight filter
Hunt Sanity:
  • "sanity <sanity>" -> Mark specified hunt sanity
  • "sanity clear <sanity>" -> Reset state of specified hunt sanity
Ghosts:
  • "ghost not <ghost>" -> Grey out the specified ghost card
  • "ghost clear <ghost>" -> Reset state of a ghost card (excluding hidden cards)
  • "ghost select <ghost>" -> Select/highlight specified ghost card
  • "ghost remove <ghost>" -> Remove the specified ghost card from the screen
  • "ghost show|info <ghost>" -> Scroll the specified ghost card into view
Timer:
  • "timer start" -> Start the smudge timer
  • "timer stop " -> Stop the smudge timer
  • "cooldown start" -> Start the cooldown timer
  • "cooldown stop " -> Stop the cooldown timer
Misc:
  • "show filters" or "show tools" -> Flips the menu to show the filters/tools
  • "ghost speed <percent>" -> Set ghost speed modifier to 50%, 75%, 100%, 125%, or 150%
  • "show map [<map name>]" -> Opens the map tab and displays map
  • "close map" or "hide map" -> Closes the map tab
  • "reset journal" or "reset cheat sheet" -> Reset the cheat sheet (will save if linked with Discord)
  • "stop listening" -> Stop the voice recognition

Datenschutz

Voice commands are routed through the built-in Web Speech API. Although microphone access is required for this feature, your voice is never recorded or stored.
