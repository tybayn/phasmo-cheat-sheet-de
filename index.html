Sprachbefehle

Mit Sprachbefehlen können Sie den Spickzettel über Ihre Stimme steuern. Sobald Sie auf „Aktivieren“ klicken, hört der Spickzettel auf Ihre Sprachbefehle und bleibt zwischen den Zurücksetzungen bestehen, bis Sie ihn deaktivieren. Es wird standardmäßig von Ihrem Standardmikrofon gelesen. Stellen Sie daher sicher, dass es richtig eingestellt ist. Die Spracherkennung funktioniert auch dann weiter, wenn die Seite nicht im Fokus ist und Sie im Spiel dasselbe Mikrofon verwenden.
Beweise:
  • "anzahl der nachweise <nummer>" oder "schwierigkeit <nummer>" -> Legen Sie die Anzahl der Beweise fest
  • "beweise <beweise>" -> Als gefundene Beweise markieren
  • "beweise nicht <beweise>" -> Als nicht gefundenen Beweis markieren
  • "beweise klar <beweise>" -> Zustand eines Beweismittels zurücksetzen
  • "affenpfote <beweise>" -> Dies wird die Beweise mit Affenpfotenlogik abgrenzen
Geschwindigkeit:
  • "geschwindigkeit <geschwindigkeit>" -> Angegebene Geschwindigkeit markieren
  • "geschwindigkeit klar <geschwindigkeit>" -> Zurücksetzen des Zustands der angegebenen Geschwindigkeit
  • "geschwindigkeit sichtlinie" -> Markierung als normale Sichtlinie
  • "geschwindigkeit nicht sichtlinie" -> Markierung als keine Sichtlinie
  • "geschwindigkeit klar sichtlinie" -> Sichtlinienfilter zurücksetzen
Jagd auf geistige Gesundheit:
  • "sanity <sanity>" -> Markierte sanity
  • "sanity klar <sanity>" -> Sanity zurücksetzen
Geister:
  • "geist nicht <geist>" -> Die angegebene Geisterkarte ausgrauen
  • "geist klar <geist>" -> Zurücksetzen des Status einer Geisterkarte (ausgenommen verdeckte Karten)
  • "geist wählen <geist>" -> Auswahl/Hervorhebung einer bestimmten Geisterkarte
  • "geist abwählen <geist>" -> Entfernen Sie die angegebene Geisterkarte vom Bildschirm
  • "geist erschau <ghost>" -> Die angegebene Geisterkarte in die Ansicht rollen
Timer:
  • "timer starten" -> Starten Sie den Timer für das Räucherstäbchen
  • "timer stoppen " -> Stoppen Sie den Timer für das Räucherstäbchen
  • "abklinkungszeit starten" -> Starten Sie den Abkühlungstimer
  • "abklinkungszeit stoppen" -> Stoppen Sie den Abkühlungstimer
Verschiedenes:
  • "filter anzeigen" oder "werkzeuge anzeigen" -> Schaltet das Menü zur Anzeige der Filter/Werkzeuge um
  • "geistergeschwindigkeit <prozent>" -> Modifikator für die Geistergeschwindigkeit auf 50%, 75%, 100%, 125% oder 150% einstellen
  • "karte [<kartenname>] anzeigen" -> Öffnet die Registerkarte Karte und zeigt die Karte an
  • "karte schließen" oder "karte ausblenden" -> Schließt die Karteikarte
  • "journal zurücksetzen" oder "cheat sheet zurücksetzen" -> Zurücksetzen der cheat sheet
  • "aufhören zuzuhören" -> Anhalten der Spracherkennung

Datenschutz

Sprachbefehle werden über die integrierte Web Speech API weitergeleitet. Obwohl für diese Funktion ein Mikrofonzugriff erforderlich ist, wird Ihre Stimme niemals aufgezeichnet oder gespeichert.
